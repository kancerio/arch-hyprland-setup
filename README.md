\# Arch Linux + Hyprland Setup



Этот репозиторий содержит скрипты и инструкции для быстрой установки и настройки системы Arch Linux с оконным менеджером Hyprland, драйверами NVIDIA, основным софтом для разработки, учёбы и инженерии.



\## Предварительные требования



\- Установленный Arch Linux с базовой системой.

\- Подключение к интернету.

\- Пользователь с правами sudo.
