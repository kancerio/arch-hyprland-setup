\# Установка 1С:Предприятие через Wine на Arch Linux



Это руководство поможет вам установить платформу 1С:Предприятие (клиентскую часть) на Arch Linux с использованием Wine. Оно основано на реальном опыте и проверено на системе с NVIDIA RTX 3060 и Hyprland.



\*\*Важно:\*\* Wine-версия 1С подходит для \*\*тонкого клиента\*\* и работы с информационными базами, но для серверной части лучше использовать \*\*нативную Linux-версию\*\* (установщик `.run`). Если у вас есть возможность — ставьте нативную, она стабильнее и быстрее.



\---



\## 📦 Подготовка системы



\### 1. Включите репозиторий `multilib` (для 32-битных библиотек)



Откройте файл `/etc/pacman.conf`:



```bash

sudo nano /etc/pacman.conf



Найдите строки:

\#\[multilib]

\#Include = /etc/pacman.d/mirrorlist



Раскомментируйте их (уберите #), чтобы стало:

\[multilib]

Include = /etc/pacman.d/mirrorlist


Сохраните и обновите базу пакетов:

sudo pacman -Sy



\### 2. Установите Wine и вспомогательные утилиты



sudo pacman -S wine winetricks wine-gecko wine-mono



\### 3. Создание Wine-префикса



export WINEPREFIX=\~/wine\_1c   # папка, где будет храниться окружение

wineboot



\### 4. Установка необходимых компонентов (.NET Framework 4.8)



winetricks --force dotnet48 corefonts



\###  Установка платформы 1С:Предприятие



1\. Скачайте дистрибутив

Зайдите на releases.1c.ru (требуется учётная запись).

В разделе «Технологическая платформа 1С:Предприятие для Windows» выберите нужную версию и скачайте файл setup.exe или 1CEnterprise\_8.3.XX.XXXX.exe.



2\. Запустите установку

Перейдите в папку со скачанным файлом и выполните:

cd \~/Downloads

wine setup.exe



Откроется графический мастер установки (почти как в Windows).

* Нажимайте «Далее», принимайте лицензионное соглашение.
* Выберите компоненты: как минимум «1С:Предприятие» и «Тонкий клиент».
* Путь оставьте по умолчанию (C:\\Program Files\\1cv8).
* Дождитесь завершения копирования файлов.



\### Настройка для стабильной работы



1\. Отключите аппаратное ускорение (если есть проблемы с графикой)

1С в Wine иногда конфликтует с драйверами NVIDIA. Если при запуске появляется чёрный экран или артефакты, сделайте следующее:

* Запустите конфигуратор Wine:



winecfg



* Перейдите на вкладку Graphics.



* Включите опцию Emulate a virtual desktop и укажите разрешение, например, 1920x1080.



* Сохраните и попробуйте запустить 1С снова.



Либо используйте переменную окружения для программного рендеринга:



export LIBGL\_ALWAYS\_SOFTWARE=1

wine "C:/Program Files/1cv8/bin/1cv8.exe"



2\. Установите дополнительные шрифты (если текст отображается квадратиками)

Если в 1С вместо текста видны прямоугольники, установите больше шрифтов:



winetricks allfonts



\### Запуск 1С

После установки запустить 1С можно командой:



wine "C:/Program Files/1cv8/bin/1cv8.exe"

* Если вы создали ярлык на рабочем столе (или в меню Wine), можно использовать его.



При первом запуске может появиться окно выбора информационной базы. Если у вас есть файл .1Cv8 или подключение к серверу, укажите их.
